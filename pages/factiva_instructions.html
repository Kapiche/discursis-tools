Factiva Export Instructions

Discursis Tools requires a very specific Factiva export format. Below are instruction on how to export articles from Factivas search results page to the correct format.

Step 1 - Do your search

Execute your Factiva search using your regular method. You should presented with a search results screen very similar to the following screenshot:

Step 2 - Select Articles

Next you need to select that articles you wish to export (or all articles):

Step 3 - Export Articles

You need to save the articles you have selected. Click the save button () and select Article Format:

Step 4 - Save the Export

The result of the previous step will be a new window with the export HTML in it. You need to save this page. In Chrome, you can right-click in the page and select Save As:

Step 5 - Generate Discursis CSV

You can now use our Factiva tool to convert the articles to a format that can be used in Discursis by upload the file you just saved!
